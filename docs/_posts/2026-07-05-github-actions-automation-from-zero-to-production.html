What Is GitHub Actions?

GitHub's built-in CI/CD service for automating software workflows — completely free for public repos. It runs on Ubuntu, Windows, and macOS runners.

Core Concepts

  • Workflow: A YAML file defining your automated process
    • Job: A collection of steps running on the same runner
    • Step: A single operation (run a command or use an action)
    • Action: A reusable custom module from the marketplace
